なにも指定しない場合は要素、hmdenkoにKUSONEMIが流れます。コード↓
//jQuery読み込み後 
<script src="../jquery-plugin/jquery.hmdenko.js"></script>
<style>
    $().loadDenko();
</style>
<div id="hmdenko"></div>


指定してやると好きな要素に文字列を流せます。コード↓
 <style>
    $().loadDenko({
        viewStr : 'TestHoge', //流す文字
        colorFill : '#0f0', //文字カラー
        colorSpace : '#030', //余白カラー
        parentElem : 'box', //要素
        moveFlg : true //true 動かす false 表示のみ
    });
</style>
